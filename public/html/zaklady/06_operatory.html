TITLE

Ruby je dynamický a vysoce flexibilní programovací jazyk, který nabízí širokou škálu operátorů pro provádění různých operací. Vedle základních aritmetických a porovnávacích operátorů nabízí Ruby také specifické operátory pro řetězce, bitové operace a podmínky.

6.1 Základní

Zde se podíváme na několik základních typů operátorů v Ruby a jak je použít. Používají se jako základní stavební kameny programovacích jazyků, které slouží k provádění různých aritmetických, logických a porovnávacích operací nad daty. Tyto operátory se běžně používají v programování pro manipulaci s hodnotami a vyhodnocování podmínek.

6.1.1 Aritmetické operátory

Aritmetické operátory představují klíčový nástroj v programování, včetně jazyka Ruby, pro provádění základních matematických operací. Tyto operátory umožňují programátorům provádět širokou škálu aritmetických operací, jako jsou sčítání, odčítání, násobení a dělení.

Kromě těchto základních operací aritmetické operátory mohou také zahrnovat operace jako je modulo (zbytek po dělení), mocnina a další. Díky aritmetickým operátorům mohou vývojáři manipulovat s čísly a provádět výpočty potřebné pro správné fungování jejich programů.

  • + (sčítání)
  • - (odčítání)
  • * (násobení)
  • / (dělení)
  • % (modulo, zbytek po dělení)
  • ** (umocňování)
a = 5
b = 2

puts a + b    # Výsledek: 7
puts a - b    # Výsledek: 3
puts a * b    # Výsledek: 10
puts a / b    # Výsledek: 2
puts a % b    # Výsledek: 1
puts a ** b   # Výsledek: 25

6.1.2 Porovnávací operátory

Porovnávací operátory slouží k porovnání hodnot různých proměnných nebo výrazů. Tyto operátory umožňují programátorům vyjádřit a řídit vztahy mezi hodnotami, jako jsou rovnost, nerovnost, větší nebo menší než, a další. Díky porovnávacím operátorům mohou vývojáři efektivně řídit chování svých programů na základě podmínek definovaných pomocí těchto porovnání.

  • == (rovnost)
  • != (nerovnost)
  • > (větší než)
  • < (menší než)
  • >= (větší nebo rovno)
  • <= (menší nebo rovno)
a = 5
b = 2

puts a == b   # Výsledek: false
puts a != b   # Výsledek: true
puts a > b    # Výsledek: true
puts a < b    # Výsledek: false
puts a >= b   # Výsledek: true
puts a <= b   # Výsledek: false

6.1.3 Logické operátory

Tyto operátory umožňují programátorům definovat vztahy mezi různými logickými výrazy a hodnotami. Pomocí logických operátorů, jako jsou AND, OR a NOT, mohou programátoři vytvářet složitější logické výrazy a podmínky, které ovlivňují chování programu.

Logické operátory jsou klíčové pro správné fungování kontrolních struktur, cyklů a podmíněných výrazů v programovacím kódu. Jsou nezbytné pro rozhodování, která část kódu se má vykonat v závislosti na splnění určitých podmínek, a tak přispívají k účinnosti a flexibilitě programů

  • && (konjunkce, logické "AND")
  • || (disjunkce, logické "OR")
  • ! (negace, logické "NOT")
a = true
b = false

puts a && b   # Výsledek: false
puts a || b   # Výsledek: true
puts !a       # Výsledek: false
puts !b       # Výsledek: true

6.1.4 Přiřazovací operátory

Tyto operátory jsou nezbytné pro manipulaci s daty v programu a umožňují programátorům ukládat a aktualizovat hodnoty v proměnných.

V jazyce Ruby je přiřazovací operátor běžně značen znakem '=', který označuje, že hodnota na pravé straně operátoru bude přiřazena proměnné na levé straně. Použití přiřazovacích operátorů je nezbytné pro správné fungování programu, protože umožňují uchování a manipulaci s daty během vykonávání kódu.

  • = (přiřazení)
  • += (sčítání a přiřazení)
  • -= (odčítání a přiřazení)
  • *= (násobení a přiřazení)
  • /= (dělení a přiřazení)
  • %= (modulo a přiřazení)
a = 5

a += 3  # Ekvivalent a = a + 3
puts a  # Výsledek: 8

a -= 3  # Ekvivalent a = a - 3
puts a  # Výsledek: 5

a *= 3  # Ekvivalent a = a * 3
puts a  # Výsledek: 15

a *= 3  # Ekvivalent a = a / 3
puts a  # Výsledek: 5

a %= 3  # Ekvivalent a = a % 3
puts a  # Výsledek: 2

6.2 Specifické

Zde se podíváme na několik specifických typů operátorů v Ruby a jak je použít. Tyto operátory se používají k provádění specifických operací, které mohou být obtížné nebo nepraktické provádět pomocí běžných aritmetických, logických nebo porovnávacích operátorů.

Tyto specifické operátory umožňují programátorům provádět různé operace v Ruby efektivněji a s méně kódovými řádky. Použití těchto operátorů může zlepšit čitelnost kódu a zkrátit délku kódu při řešení určitých problémů.

6.2.1 Operátory pro práci s řetězci

Tyto operátory jsou často používané při manipulaci s řetězci znaků v programovacím kódu.

Například, spojovací operátor slouží k připojení dvou řetězců do jednoho, zatímco operátor pro opakování umožňuje vytvořit řetězec opakovaním určitého znaku nebo řetězce. Formátovací operátory umožňují programátorům definovat speciální vzory pro formátování textu, což je užitečné při generování strukturovaného výstupu, jako jsou tabulky nebo zprávy.

Tato funkcionalita je často potřebná při práci s textovými daty v programu a přispívá k jeho flexibilitě a schopnosti manipulovat s textem podle potřeby

  • + (řetězcová konkatenace)
  • * (opakování řetězce)
  • % (formátování řetězce)
str1 = "Hello, "
str2 = "world!"

puts str1 + str2    # Výsledek: Hello, world!
puts str1 * 3       # Výsledek: Hello, Hello, Hello, 
puts "%s %s" % [str1, str2]  # Výsledek: Hello, world!

6.2.2 Bitové operátory

Bitové operátory představují důležitý nástroj v jazyce Ruby pro manipulaci s binárními hodnotami na úrovni jednotlivých bitů. Ačkoli mohou na první pohled působit exoticky a méně obvykle, mají široké spektrum praktických využití v různých oblastech softwarového inženýrství.

Jedním z hlavních využití je manipulace s příznaky (flags), což jsou jednotlivé bity v paměti, které představují určité stavy nebo nastavení v programu. Díky bitovým operátorům je možné efektivně nastavovat, resetovat a testovat tyto příznaky. Dalším využitím je oblast šifrování, kde se bitové operátory často používají pro provádění šifrovacích a dešifrovacích operací na úrovni jednotlivých bitů. Kromě toho mohou být bitové operátory využity i pro optimalizaci kódu, kde umožňují provádět efektivní a kompaktní manipulace s daty na binární úrovni.

Celkově lze říci, že bitové operátory jsou mocným nástrojem, který přispívá k flexibilitě a výkonnosti aplikací napsaných v jazyce Ruby a umožňuje programátorům provádět sofistikované operace s binárními daty.

  • & (bitový and)
  • | (bitový or)
  • ^ (bitový xor)
  • ~ (bitová negace)
  • << (bitový posun doleva)
  • >> (bitový posun doprava)
a = 5  # Binárně: 101
b = 3  # Binárně: 011

puts a & b   # Výsledek: 1 (binárně: 001)
puts a | b   # Výsledek: 7 (binárně: 111)
puts a ^ b   # Výsledek: 6 (binárně: 110)
puts ~a      # Výsledek: -6 (binárně: 111...1010)
puts a << 1  # Výsledek: 10 (binárně: 1010)
puts b >> 1  # Výsledek: 1 (binárně: 01)

6.2.3 Operátor třícestného podmínění

Tento speciální operátor je často používán v programování pro vytvoření jednoduchých podmíněných výrazů. Jeho využití umožňuje programátorům zapsat podmíněný výraz do jednoho řádku kódu, což může vést k výrazně čistějšímu a méně zdlouhavému kódu.

Tím, že podmíněný výraz zahrnuje logický test a dvě možné hodnoty, které se mají vrátit podle toho, zda je test vyhodnocen jako pravdivý či nepravdivý, je možné vyhnout se zbytečnému použití rozsáhlých podmíněných bloků. To vede k elegantnějšímu a snadněji čitelnému kódu, což je důležité pro udržitelnost a správnou údržbu programu v dlouhodobém horizontu.

  • ? : (ternární operátor)
x = 10
puts x > 5 ? "yes" : "no"  # Výsledek: yes

6.2.4 Operátor společnosti

Operátor společnosti je často využíván v situacích, kdy je potřeba provádět porovnání mezi objekty různých typů, jako jsou čísla, řetězce, objekty tříd atd. Vrací jednu z tří hodnot: -1, 0 nebo 1, což označuje, zda je levá hodnota menší, rovná se, nebo je větší než pravá hodnota.

  • <=> (spaceship operátor)
puts 5 <=> 10   # Výsledek: -1
puts 10 <=> 10  # Výsledek: 0
puts 10 <=> 5   # Výsledek: 1

6.2.5 Operátor vytváření rozsahu

Operátor rozsahu je často využíván v programování k definici sekvencí hodnot mezi dvěma určenými body. Jeho použití umožňuje programátorům jednoduše vytvořit a reprezentovat rozsah hodnot, které mají být zahrnuty v určité sekvenci.

Tento operátor je obvykle značen dvěma tečkami (..) nebo třemi tečkami (...), které označují počáteční a koncovou hodnotu rozsahu. Pokud jsou tyto dvě hodnoty celočíselné, vytváří operátor rozsahu sekvenci všech celých čísel mezi nimi. V případě, že jsou hodnoty znaky nebo jiné datové typy, operátor rozsahu vytváří sekvenci těchto hodnot podle abecedního pořadí nebo specifikovaného porovnávacího kritéria.

Použití tohoto operátoru je užitečné zejména při definování iterací, procházení kolekcí a generování sekvencí hodnot v programování, což přispívá k čitelnosti, efektivitě a údržbě kódu.

  • .. (dvojitá tečka): Vytváří rozsah, který zahrnuje počáteční a koncovou hodnotu.
  • ... (trojitá tečka): Vytváří rozsah, který zahrnuje počáteční hodnotu, ale nezahrnuje koncovou hodnotu.
range_inclusive = (1..5)
puts range_inclusive.to_a.inspect  # Výstup: [1, 2, 3, 4, 5]

range_exclusive = (1...5)
puts range_exclusive.to_a.inspect  # Výstup: [1, 2, 3, 4]
